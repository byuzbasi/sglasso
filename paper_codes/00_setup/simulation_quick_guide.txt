SGLASSO SIMULATION SERVER QUICK GUIDE
====================================

Use the unified guide in:

00_setup/server_run_guide.txt

Current simulation outputs are organized under:

results/
├── main_simulation/
├── tuning_sensitivity/
└── scalability/

Submit from the project root:

sbatch paper_codes/02_main_simulation/run_sim_slasso_all.sh
sbatch paper_codes/03_tuning_sensitivity/run_sglasso_tuning_sensitivity.sh
sbatch paper_codes/04_scalability/run_scalability_analysis.sh

Small tests:

REPEATNUM=1 OUTDIR=results/main_simulation_test \
  Rscript --vanilla paper_codes/02_main_simulation/run_sim_slasso.all.R

Rscript --vanilla paper_codes/03_tuning_sensitivity/run_sglasso_tuning_sensitivity.R \
  --nrep=1 \
  --cores=1 \
  --outdir=results/tuning_sensitivity_test

Rscript --vanilla paper_codes/04_scalability/run_scalability_analysis.R \
  --p_grid=1000 \
  --J_grid=200 \
  --nrep=1 \
  --nlambda=3 \
  --outdir=results/scalability_test

Main output files:
